# 개인 통합 점검 체크리스트 (공유 전)

트렐로 카드에서 체크리스트를 만들 때, 아래 블록을 "체크리스트 이름"으로 만들고
해당 항목들을 줄단위로 붙여넣으면 한 줄씩 깔끔하게 분리됩니다.

---

## 체크리스트명: 0) 범위 고정 (팀 영향 최소화)
- 이번 반영 범위를 명확히 정함 (예: `Assets/2.SSH`만)
- 범위 밖 파일(`1.Jsr`, `3.JH`, `4.SDH`)이 커밋에 섞이지 않음
- 삭제 파일이 의도된 삭제인지 확인함
- `.meta` 이동/변경이 의도된 것인지 확인함
- 씬 파일 변경이 꼭 필요한 것만 포함됨

## 체크리스트명: 1) 게임오버/리스폰/리트라이
- 플레이어 피격 시 목숨 UI가 1칸 감소함
- 목숨 0일 때 GAME OVER 패널이 표시됨
- Retry 버튼 클릭 시 현재 씬이 정상 재시작됨
- 재시작 후 목숨/점수/폭탄 UI가 초기값으로 복구됨
- 리스폰 대기시간 후 플레이어가 정상 위치에 복귀함

## 체크리스트명: 2) 점수 시스템
- 적 처치 시 점수가 즉시 증가함
- 적 타입별 점수(A/B/C)가 의도값으로 반영됨
- 코인 획득 시 점수 증가가 반영됨
- 점수 텍스트가 전투 중 실시간 갱신됨
- 게임 재시작 시 점수가 초기화됨

## 체크리스트명: 3) 아이템 시스템
- 아이템 드랍이 정상적으로 발생함
- 코인 아이템 충돌 시 정상 획득됨
- 파워 아이템 획득 시 화력/추종자 증가가 반영됨
- 붐 아이템 획득 시 폭탄 UI가 증가함
- 아이템 획득 후 오브젝트가 정상 제거됨

## 체크리스트명: 4) 폭탄(붐) 시스템
- 폭탄 최대치가 3으로 제한됨
- 폭탄 사용 입력 키가 정상 동작함
- 폭탄 사용 시 폭탄 UI가 1 감소함
- 폭탄 사용 시 화면 내 적이 제거됨
- 폭탄 사용 시 화면 내 적탄이 제거됨
- 폭탄으로 제거된 적도 점수에 반영됨
- 폭탄 0개일 때 사용이 차단됨(오동작 없음)

## 체크리스트명: 5) UI 연동
- 씬에 UIManager 오브젝트가 1개만 존재함
- UIManager 참조(images/gameOver/retry/score/boom)가 유효함
- 자동 바인딩 사용 시 오브젝트 이름이 규칙과 일치함
- Life/Boom 이미지 alpha 표시가 의도대로 동작함
- GAME OVER 패널 표시/숨김 전환이 정상임

## 체크리스트명: 6) 태그/충돌/입력
- Player 태그가 정확함
- Enemy/EnemyBullet 태그가 정확함
- 아이템 충돌 레이어/트리거 설정이 맞음
- 피격 시 중복 데미지(한 번에 과다 차감) 없음
- 입력 키(공격/폭탄)가 서로 충돌하지 않음

## 체크리스트명: 7) 풀링/성능
- 적/총알 비활성화 후 재사용이 정상 동작함
- 풀 반환 시 체력/스프라이트 상태가 초기화됨
- 장시간 플레이 시 오브젝트 누적/누수 없음
- 폭탄 연속 사용 시 프레임 급락 없음

## 체크리스트명: 8) 씬/프리팹 안정성
- 씬 이름 변경(Main/Test 등)이 팀 합의와 일치함
- UI 프리팹 변경 후 씬 참조가 깨지지 않음
- 프리팹 적용 후 불필요한 오버라이드가 없음
- EventSystem/Canvas 등 필수 UI 오브젝트 누락 없음

## 체크리스트명: 9) Git 최종 점검 (개인용)
- `git status`로 변경 파일 최종 확인함
- 의도한 경로만 `git add` 함
- 커밋 메시지에 반영 범위를 명시함
- push 전 pull/rebase로 충돌 여부 확인함
- push 후 로컬 잔여 변경(stash/미커밋) 상태 확인함

## 체크리스트명: 테스트 기록
- 날짜 기록
- 테스트 씬 기록
- 빌드/에디터 기록
- 확인자(본인) 기록
- 메모 작성
